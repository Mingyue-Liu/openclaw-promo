5 分钟快速开始

简单 4 步，拥有你的 AI 助手

1

安装 Node.js 和 pnpm

# 安装 Node.js (v18+) brew install node # macOS # 或从 https://nodejs.org 下载 # 安装 pnpm npm install -g pnpm

确保 Node.js 版本 >= 18，pnpm 是更快的包管理器

2

安装 OpenClaw

# 安装 OpenClaw pnpm add -g openclaw # 验证安装 openclaw --version

全局安装 OpenClaw CLI 工具

3

配置 API Key

# 启动配置向导 openclaw configure # 或手动配置模型 # 推荐使用 Dashscope (通义千问) # https://dashscope.console.aliyun.com/

配置 AI 模型 API Key，推荐使用阿里云 Dashscope（有免费额度）

4

启动 Gateway 并开始使用

# 启动 Gateway 服务 openclaw gateway start # 查看状态 openclaw gateway status # 开始对话 # 通过钉钉/微信/Telegram 或 Web 界面

Gateway 启动后，你就可以通过消息或 Web 界面与 AI 交互了！
